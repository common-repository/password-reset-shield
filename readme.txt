== Description ==

Password Reset Shield makes easy to disable your password recovery options on WordPress Site, also it allows the admin to track details who requested password reset with various information such as users IP, Browser and Date/Time.

* Password Reset/Recovery Shield
* Disable or Enable password recovery options
* Disable or Enable information gathering on password recovery page

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Enjoy!
